Тип шрифта на CSS

Свойство font family определяет семейтсво шрифта

Самые распространненые шрифты это: Times New Roman (стоит по умолчанию), Arial, Georgia, Verdana, Trebuchet MS, Impact, Arial Black, Comic Sans MS, Courier New.

Тут будет шрифт Arial потому что я ему задам его с помощью font-family Arial

Если шрифт состоит из нескольких слов то имя такого шрифта следует брать в ковычки вот так

'Comic Sans MS'

Межстрочный интервал CSS

Это расстояние между линями текста, то есть белый интервал между словами. Пример ниже

Lorem ipsum dolor sit amet consectetur adipisicing elit. Aliquid inventore soluta odio culpa repellat similique cumque tempora fugit possimus ducimus!

В тексте выше lh будет равняться 50 пикселям

Значение lh можно указывать без единиц. То есть не обязательно должно быть целое число. МОжно указать его так line-height: 1.5

Свойство сокращение font

В свойстве font указывается сразу несколько значений для текста, и оно содержит в себе размер шрифта, семеймтво, жирность, курсив, и межстрочный интервал

Порядок свойств имеет значение. Есть обязательные свойства размер шрифта, семейтсво.

У этого текста шрифт будет в 16 пикселей и семейство будет Arial потому что я укажу font: 16px Arial

Красная строка текста на CSS

Для того что бы указать красную строку. Есть свойство text-indent.

Lorem ipsum dolor sit amet consectetur, adipisicing elit. Voluptatem, assumenda. Тут будет красная строка потому что мы ввели text-ident 50px
